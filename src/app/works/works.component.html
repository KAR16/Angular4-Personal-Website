CLIENT APP

For This Project, I created Two Different Apps. The Client App and The Driver App. You Access The App By First Creating An Account, Adding A Phone Number, Then Using Gps Technology On Google Maps Api, Your Location Is Automatically Selected, But You Could Point The Marker Elsewhere To Choose Another Location. I completed The Entire Project In A month and 3 weeks. Check Out The Open Source On Github

DRIVER APP

This is The Driver Version Of The App, Works With Onesignal device notification and google maps to make it easy for the driver to locate the client. The app recieves a notification from the client when set to active. Then when accepted, The client's details are displayed for the best experience. Check Out The Open Source On Github

DASHBOARD

With The Dashboard The Team could see how many clients are signing up, accept the drivers after going through their credentials and also make changes to the company with the stats they collect. It Was Built With Angular4+ and Firebase as overall Database For The Entire Project. Check Out The Open Source On Github
